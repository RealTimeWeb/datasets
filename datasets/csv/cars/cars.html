Field Descriptions

Key List of... Comment Example Value
City mpg Integer number The miles-per-gallon this car gets on average in cities. 18
Classification String Whether this is a "Manual transmission" or an "Automatic transmission". If it is unknown, it is left blank. "Automatic transmission"
Driveline String A string representing whether this is "Rear-wheel drive", "Front-wheel drive", or "All-wheel drive". "All-wheel drive"
Engine Type String How many cylinders are in this engine. Most cars are either a 6-cylinder or an 8-cylinder. "Audi 3.2L 6 cylinder 250hp 236ft-lbs"
Fuel Type String Whether this car takes "Gasoline", "Diesel fuel", "Electricity", "Compressed natural gas", or "E85" (a term that refers to high-level ethanol-gasoline blends containing 51%-83% ethanol). If it is unknown, it is left blank. "Gasoline"
Height Integer number Unknown values are stored as 0. Unfortunately, many cars do not report this data. 140
Highway mpg Integer number The miles-per-gallon this car gets on average on highways. 25
Horsepower Integer number A measure of the engine's power. A unit of power equal to 550 foot-pounds per second (745.7 watts). 250
Hybrid Integer number Whether this is a hybrid engine or not - that is, if it uses both an internal combustion engine and an electric motor. False
ID String A unique ID for this particular car, using the year, make, model, and transmission type. "2009 Audi A3 3.2"
Length Integer number Unknown values are stored as 0. Unfortunately, many cars do not report this data. 143
Make String The maker for this car. "Audi"
Model Year String The specific name/year for this car. "2009 Audi A3"
Number of Forward Gears Integer number The number of forward gears. If no information is available, it is coded as 0. 6
Torque Integer number The torque of the engine, measured in lb/ft. When an engine is said to make "200 lb-ft of torque", it means that 200 pounds of force on a 1-foot lever is needed to stop its motion. 236
Transmission String The full name of this type of transmission, based on its Classification and number of forward gears. "6 Speed Automatic Select Shift"
Width Integer number Unknown values are stored as 0. Unfortunately, many cars do not report this data. 202
Year Integer number The year that this car was released. 2009
