Overview

The Current Population Survey (CPS) is a monthly survey of households conducted by the Bureau of Census for the Bureau of Labor Statistics. It provides a comprehensive body of data on the labor force, employment, unemployment, persons not in the labor force, hours of work, earnings, and other demographic and labor force characteristics.

http://www.bls.gov/cps/home.htm

Usage

Begin by adding the above JAR file to your classpath. This library has 1 function you can use. 
import corgis.labor.LaborLibrary;
import corgis.labor.domain.*;
import java.util.ArrayList;

public class ExampleClass {
    public static void main(String[] args) {
        // Get access to the library
        LaborLibrary laborLibrary = new LaborLibrary();
        // Access data inside the library
        ArrayList<Result> list_of_result = laborLibrary.getResults();
    }
}
